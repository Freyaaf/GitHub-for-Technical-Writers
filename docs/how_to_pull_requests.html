4.1.2 如何拉取请求

  1. fork 项目
    1. 选择一个你要提交 pull request 的项目，以onelasttry为例。
    2. 点击右上方的Fork按钮。
    3. 选择Create fork。
    4. 可以看到这个repository已经被成功fork下来。
  2. 加入内容
    1. 点击铅笔按钮，进入编辑模式。
    2. 按照格式加入你想加入的内容。
    3. 输入Commit信息，点击下面的Commit changes按钮。
    4. 可以看到，你fork出来的repository中，内容已经发生改变。
  3. 发送Pull Request
    1. 点击你fork的仓库中的Pull request按钮。接下来的页面会显示你修改的内容，确认无误后点击Create pull request按钮。
    2. 接下来添加一些你的pull request的描述，保证你的pr可以让原repository同意接受，等待对方处理即可。
  4. 同意Pull Request
    1. 查看Pull requests。
    2. 审查无误后，可以merge这个pull request到自己的仓库中，点击Merge pull request按钮。
    3. 点击Conform merge。
    4. 可以看到，pull request的内容已经被合并到原repository中。
