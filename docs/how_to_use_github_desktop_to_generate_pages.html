使用Github Desktop添加文件并发布网页

  1. 安装GitHub Desktop：从 GitHub Desktop 官方页面 下载并安装。
  2. 克隆仓库
    1. 打开你想复制的repository，点击Code，选择Open with GitHub Desktop。
    2. 打开 GitHub Desktop，登录账户。
    3. 点击“Clone a repository”。
    4. 选择仓库并克隆到本地。
  3. 添加文件
    1. 在本地仓库文件夹新建[docs]文件夹和[ditasrc]文件夹。
    2. 复制除了[out]文件夹之外的所有文档到[ditasrc]文件夹。
    3. 把[out]文件夹内的[webhelp]文件夹所有文档复制到[docs]文件夹。
    4. 在 GitHub Desktop 中，选择“Changes”标签。在左下角聊天框输入：add dita source and webhelp。点击“Commit to main”。
  4. 推送更改：点击“Push Origin”将更改同步到远程仓库。
  5. 启用 GitHub Pages
    1. 回到GitHub网页，打开仓库，打开docs文件夹。
    2. 点“Settings”，在“Pages”选项下，选择分支（main，docs），点击“Save”。
    3. 等待几秒后刷新网页即可，网页将在https://username.github.io/repository-name/发布。
